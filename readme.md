# Uruchomienie
## backend
 w folderze back/backend:
 
 `docker compose up`
 
 `symfony console doctrine:migrations:migrate`

 `symfony local:server:start`

 ## frontend
 w folderze front/frontend

 `npm install`

 `npm run dev`

 # Użyte zewnętrzne biblioteki
  - doctrine/orm
  - nelmio/cors-bundle
  - tailwindcss
  - axios
